# BioVote: AI-Powered OTP-Based Digital Voting System

## Overview

BioVote is a secure online voting platform designed to ensure transparent and reliable digital elections.
The system integrates **AI-based face recognition**, **liveness detection**, and **OTP authentication** to verify voter identity and prevent fraudulent voting activities.

This project aims to enforce the **One-Person-One-Vote principle** while providing a secure and user-friendly online voting experience.

---

## Key Features

* Secure voter authentication using **Face Recognition**
* **Liveness Detection** to prevent photo or spoofing attacks
* **OTP-based verification** for additional security
* **Admin dashboard** to manage candidates and voters
* Ensures **One-Person-One-Vote rule**
* Secure vote recording and storage

---

## System Architecture

The system consists of the following components:

1. **Frontend**

   * Developed using React
   * Provides user interface for voters and administrators

2. **Backend**

   * Developed using Flask
   * Handles authentication, vote processing, and database interaction

3. **AI Module**

   * Face recognition for voter identity verification
   * Liveness detection to prevent spoof attacks

4. **OTP Service**

   * Twilio-based OTP verification for additional authentication

---

## Technologies Used

* **Frontend:** React, JavaScript, HTML, CSS
* **Backend:** Python, Flask
